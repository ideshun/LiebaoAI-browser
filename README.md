- 书签不能正常同步，书签拖动排序闪退

- 播放音频的标签，点击小喇叭无法静音
- AI助手，点击停止生成 还在继续输出（非必现）
- 部分网页卡死
- 猎豹AI导航新标签页天气，在未选择城市的时候显示异常
- i.liebao.cn 在新版中打不开，无法恢复历史书签
- ...
